Материнская плата

Материнская плата (motherboard) — это центральный элемент компьютера, который связывает все остальные компоненты системы и определяет её возможности. Она включает в себя печатную плату с чипсетом, слотами для оперативной памяти, расширения, портами подключения и системами охлаждения


1. Форм-факторы:

• ATX: Подходит для мощных игровых и профессиональных систем, много места для расширений

• Micro-ATX: Компактнее, баланс между размером и расширяемостью, для домашних и офисных ПК

• Mini-ITX: Компактные сборки, ограниченное количество слотов расширения


2. Сокет: Выбор сокета важен для совместимости с процессором

• Intel: Использует LGA (Land Grid Array)

• AMD: Использует PGA (Pin Grid Array)


3. Оперативная память:

• DDR4: Для большинства современных задач

• DDR5: Для высокопроизводительных сборок, большая скорость передачи данных

• Учитывать частоту и объём памяти (16 ГБ для игр и работы, 32 ГБ и выше для требовательных задач)


4. Слоты расширения:

• PCI Express (PCIe): Основной слот для видеокарт и других устройств


5. Порты подключения:

• USB: Проверьте количество и версии портов

• SATA: Для подключения HDD и SSD

• Аудио и сетевые порты: Для подключения периферии и стабильного интернета


6. Системы охлаждения:

• Наличие радиаторов на чипсете и VRM

• Поддержка водяного охлаждения


7. Функции для разгона:

• Если планируется разгон, стоит выбирать плату с улучшенной подсистемой питания и настройками


8. Определение чипсета:

8.1 H-чипсет:

• Обычно используется в высокопроизводительных системах

• Обеспечивает поддержку для мощных процессоров и большого объема оперативной памяти

• Часто включает в себя продвинутые функции для гейминга и профессиональных приложений


8.2 B-чипсет:

• Предназначен для среднего уровня пользователей и геймеров

• Обеспечивает баланс между производительностью и стоимостью

• Поддерживает большинство современных приложений и игр, но с меньшим количеством функций, чем H-чипсеты


8.3 Z-чипсет:

• Используется в системах с поддержкой разгона (overclocking)

• Предлагает расширенные возможности для настройки и оптимизации системы

• Часто включает в себя дополнительные функции для улучшения производительности, такие как улучшенная поддержка памяти и расширенные возможности BIOS
